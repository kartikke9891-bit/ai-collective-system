## 📦 Monorepo Structure

### 4 Project Nodes
- **King AI Voice** (LEADER) - Master control hub
- **Sovereign Aura Nexus** (NODE 2) - Follower node
- **Omega Flux Studio** (NODE 3) - Follower node
- **Royal Scribe Studio** (NODE 4) - Follower node

### Global Configuration System
- Unified design system (colors, fonts, spacing)
- Feature flags for all 4 nodes
- Global config management
- Design tokens library

### Command & Sync Infrastructure
- Command propagation (Leader → All nodes)
- Cross-node audit system
- Health monitoring
- Global config sync

### CI/CD Automation
- Automatic sync every 6 hours
- Health checks every hour
- Audit on every code push

### Memory & Persistence
- Command execution logs
- Deployment history
- User preferences
- Node status tracking

## 🚀 Quick Start

```bash
# Install dependencies
pnpm install

# Sync all nodes with global config
pnpm run sync:collective

# Run health check on all 4 nodes
pnpm run health:check

# Issue a command to propagate
pnpm run command:propagate "Update feature across all nodes"

# Run cross-node audit
pnpm run audit:collective
```
